# 우분투에 톰캣9 설치하기

## 순서
1. tomcat 9 설치
2. `톰캣홈/webapps/` war 파일을 톰캣홈으로 이동
3. mysql dumps.sql 파일을 읽기

  `mysql db_name < text_file`

4. 톰캣 실행
- https://tecadmin.net/install-tomcat-9-on-ubuntu/
