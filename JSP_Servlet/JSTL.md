# JSTL
Java Server Pages Standard Tag Library. 표준화된 태그 라이브러리. core, formatting, sql, xml, functions 기능별로 존재한다.

## 다운로드 및 설치
1. 톰캣 공식 홈페이지에서 taglibs 메뉴에서 다운로드
2. 웹프로젝트경로/WebContent/WEB-INF/lib 에 붙여넣기

## 종류
### core
- c:set
- c:out
- c:remove
- c:catch
- c:if
- c:choose
- c:forEach
- c:forTokens
- c:import
- c:url
- c:redirect
